What is this, exactly?

YOUR OWN
PERSONAL
INTERNET BOUNCER

Every time you visit a website, your device first sends a DNS query — essentially asking "hey, what's the IP address for this domain?" DNS Filter sits at that exact moment and decides whether your request gets through or gets bounced.

It's like having a very opinionated bouncer at the door of the internet. Malware sites? No entry. Adult content? Not tonight. That gambling site you've been stress-visiting? Sir, this is a Wendy's.

Unlike browser extensions (which only work in one browser and can be bypassed with a second browser), DNS filtering works at the network level — it covers every app, every browser, and every device you point at it. There is no sneaking past it.

It's written in Go, which means it's fast, lean, and doesn't need a Kubernetes cluster to run. SQLite stores everything locally. No subscriptions, no cloud accounts, no sending your browsing habits to a third party. You run it. You own it.

How a query flows
01
Browser makes DNS request
You type "example.com" and hit Enter. Your OS fires a DNS query to 127.0.0.1 (that's us).
02
Filter checks the domain
DNS Filter checks the domain against your active blocklists, categories, and whitelist — in microseconds.
✗
Blocked? Returns 0.0.0.0
The domain resolves to nothing. The browser gets a connection error. The site never loads.
✓
Allowed? Forwards upstream
Legitimate domains are forwarded to your upstream DNS (e.g. 8.8.8.8) and the real IP is returned.
Built-in capabilities

EVERYTHING
OUT OF THE BOX

High-Performance Core

Pure Go DNS server that handles thousands of queries per second. It won't become the bottleneck in your stack — your browser's tab hoarder habits will.

Category Blocking

Block entire content categories — adult, malware, gambling, ads — with a single config toggle. Curated lists included. Custom lists supported.

Web Dashboard

Manage everything from a clean web UI at localhost:8080. View stats, toggle blocklists, manage whitelists — no terminal needed after setup.

Real-Time Analytics

See exactly what's getting blocked, when, and how often. Full query logs in SQLite. No cloud, no analytics platform, no VC-funded dashboard SaaS.

Schedule-Based Rules

Block certain categories only during work hours, school hours, or that 2am doom-scrolling window you've been meaning to close off.

REST API

Full programmatic control. Integrate DNS Filter into your homelab, automation scripts, or whatever overcomplicated system you're building this weekend.
